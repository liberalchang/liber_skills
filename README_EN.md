# liber_skills

A collection of personal skills. Each skill is independently maintained with its own `SKILL.md` documentation and scripts/resources.

**Repository URL**: https://github.com/liberalchang/liber_skills.git

**Project Description**: Personal collection of developed skills

## Skills List

### frontendslides

- **Name**: `frontend-slides`
- **Description**: Create stunning, animation-rich HTML presentations from scratch or by converting PowerPoint files. Use when the user wants to build a presentation, convert a PPT/PPTX to web, or create slides for a talk/pitch. Helps non-designers discover their aesthetic through visual exploration rather than abstract choices.
- **Version**: `1.0.0`
- **Directory**: `frontendslides/`

### barkpush

- **Name**: `bark-push`
- **Description**: Smart Bark push assistant with multi-user management, intelligent content recognition, history tracking, and message update functionality
- **Version**: `1.2.0`
- **Directory**: `barkpush/`

### nano-banana-pro-openrouter

- **Name**: `nano-banana-pro`
- **Description**: Generate/edit images with Nano Banana Pro (Gemini 3 Pro Image) via OpenRouter. Use for image create/modify requests incl. edits. Supports text-to-image + image-to-image; 1K/2K/4K; use --input-image.
- **Version**: `v1.0.0`
- **Directory**: `nano-banana-pro-openrouter/`

### liber-speechapi

- **Name**: `liber-speechapi`
- **Description**: Use Liber SpeechAPI for three speech workflows: (1) Telegram/openclaw voice-message handling with ASR, concise reply summarization, and Telegram-compatible OGG/Opus TTS, (2) direct text-to-speech generation when the user explicitly asks to synthesize text into audio, and (3) direct speech-to-text transcription when the user explicitly asks to convert audio into text or structured JSON. Support environment-based configuration, optional voice cloning from a reference audio file, and fallback between shared python-env skill and the local Python environment.
- **Version**: `1.0.0`
- **Directory**: `liber-speechapi/`

### html2pptx

- **Name**: `html2pptx`
- **Description**: Convert HTML presentations to PowerPoint (PPTX) format. Supports HTML generated by frontend-slides, preserving structure, styles, and content. Trigger words: html to pptx, html2pptx, convert ppt, generate pptx
- **Version**: `3.0.0`
- **Directory**: `html2pptx/`

## Installation Methods (Reference Clawhub)

### Method A: Install from Clawhub Store (After Publishing)

- **Step 1**: Install/Configure Clawhub (or OpenClaw) client
- **Step 2**: Use `clawhub search <skill-name>` to search for skills, then use `clawhub install <skill-name>` to install
- **Step 3**: Complete necessary configuration according to each skill's `SKILL.md` (e.g., environment variables, configuration files, state directories, etc.)

### Method B: Local Installation (Current Repository)

- **Step 1**: Clone this repository locally
- **Step 2**: Copy/link the required skill directory to your skills directory (e.g., `~/.openclaw/skills/`)
- **Step 3**: Navigate to the corresponding directory and read `SKILL.md`, then run scripts or commands as instructed
